umx: Welcome to getting structural equation modeling done fast.

umx is an R package relentlessly focussed on making Structural Equation Modeling) in OpenMx quicker, easier, and more powerful.

It's easy to learn and easy to get print-ready results from models. Sweet huh? Installation is easy (look left)

Install umx:

{% highlight R %} # install & load newish devtools install.packages("devtools") library("devtools") # install and load umx install_github("tbates/umx") library("umx") # get going :-) ?umx {% endhighlight %}

On windows you might need

{% highlight R %} install_github("umx", username = "tbates", args = "--no-multiarch") {% endhighlight %}

on old versions of devtools you might need

{% highlight R %} install_github("umx", username = "tbates") {% endhighlight %}

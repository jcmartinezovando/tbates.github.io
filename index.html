Welcome to getting structural equation modeling done fast.

umx is an R package with one purpose: helping you get things done with Structural Equation Modeling using OpenMx. The design goals are that umx should make models things quicker and easier to implement, update, and publish.

umx is easy to learn, and too remember: Follow the tutorial get print-ready results in a few minutes. Sweet huh? Installation is easy (look at the install post below!)
